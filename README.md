🍣 Sushi Restaurant Management System
📌 Project Overview

This project is a Sushi Restaurant Management System built entirely in Java (without Spring Boot).
It is currently in the initial development phase, focusing on core structure, basic functionality, and foundational design.

The goal of this project is to simulate and manage the operations of a sushi restaurant, including orders, menu management, and customer interactions.


🚧 Current Status

⚠️ This project is in its early stage (initial phase).
Features are being actively designed and implemented.
